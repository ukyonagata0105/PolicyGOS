# 公開にあたっての注意

## このリポジトリの位置づけ

PolicyGOS は、行政 PDF をもとに briefing を生成するためのローカル実行型 OSS です。

この公開版では、以下を対象にしています。

- prompt-first / PDF 添付型の frontend
- OCR / repair / source discovery を担う backend
- ローカル開発・検証フロー

## まだ改善中の点

- 評価指標の実数値抽出は PDF の構造に大きく依存します
- スキャン PDF の表抽出は、今後も継続改善が必要です
- 一部の内部名称に legacy 由来の名残があります

## 期待する使い方

- 行政資料を住民向け briefing に整理する
- 政策 PDF を質問起点で読み解く
- PDF を添付して要点整理ページを生成する

## 非対象

- SaaS 提供
- 完全な動作保証
- すべての PDF レイアウトへの一律対応
