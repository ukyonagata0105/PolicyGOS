# PolicyGOS 概要

PolicyGOS は、行政 PDF から briefing を生成するためのローカル実行型 OSS です。

![PolicyGOS prompt-first shell](images/prompt-first-shell.png)

## 想定する利用者

- 行政資料を住民向けに説明したい人
- 政策文書の論点や評価指標を整理したい人
- PDF を添付して質問から説明ページを作りたい人

## 主な機能

- prompt-first の composer UI
- PDF 添付による briefing 起動
- OCR / repair / source discovery
- browser / fullscreen / ZIP export

## 位置づけ

この OSS は、政策文書をそのまま読むのではなく、質問と PDF を起点に説明面を生成するためのツールです。
